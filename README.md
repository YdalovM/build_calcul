# Build Calcul

Минимальная база проекта под новый продукт DayLab про строительные калькуляторы.

## Запуск

```bash
yarn dev
```

Открыть в браузере: [http://localhost:3000](http://localhost:3000)

## Прод-сборка

```bash
yarn build
yarn start
```

Перед продом задайте `NEXT_PUBLIC_SITE_URL` (см. `.env.example`), чтобы
canonical/sitemap/schema использовали реальный домен.

## Адаптив (обязательно)

- Проект поддерживает mobile-first сценарий: мобильный хедер + меню на телефонах, сайдбар на `md+`.
- Целевые ширины проверки: `320`, `375`, `390`, `414`, `768`, `1024` px.
- Для интерактивных элементов используется увеличенный tap-target (около `44px`).
